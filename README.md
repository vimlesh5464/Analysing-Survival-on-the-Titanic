## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset
using the ydata-profiling library to understand data quality, distributions,
missing values, correlations, and insights relevant for machine learning.

## Files Included
- Titanic-Dataset.csv : Original dataset
- titanic_profiling.html : Auto-generated EDA profiling report
- eda_notebook.ipynb : (Optional) Notebook used to generate the report
